# 源码阅读笔记

本目录用于存放 Xen 源码阅读过程中的笔记和总结。

## 目录结构建议

- `xen-hypervisor/` - Xen Hypervisor 相关笔记
- `lixom/` - Lixom 项目相关笔记
- `concepts/` - 概念和架构理解
- `templates/` - 笔记模板

## 笔记格式建议

每篇笔记建议包含以下内容：
- **日期**：记录阅读日期
- **文件路径**：相关源码文件路径
- **核心内容**：关键代码片段和解释
- **理解总结**：个人理解和思考
- **疑问**：遇到的问题和待解答的问题
- **参考链接**：相关文档和资源

可以参考 `example-note.md` 作为笔记模板。
